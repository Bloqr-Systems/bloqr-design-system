A windowed code block — navy surface, mac traffic-light dots, optional filename. "You'll never write this, but it's here if you want it."

```jsx
<CodeWindow filename="example.ts">
  <span style={{ color: 'var(--code-comment)' }}>// Install: deno add @bloqr/compiler</span>{'\n'}
  <span style={{ color: 'var(--code-keyword)' }}>import</span> {'{ compile }'} ...
</CodeWindow>
```

Syntax palette tokens: `--code-keyword` (violet), `--code-type`/`--code-string`/`--code-number`, `--code-fn` (amber), `--code-comment` (slate), `--code-accent` (orange for enum members / HTTP verbs).
