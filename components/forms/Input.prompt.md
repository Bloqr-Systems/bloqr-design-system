Standard text field — navy surface, slate border that turns orange on focus. Includes label, hint, and error states.

```jsx
<Input label="Email" placeholder="you@example.com" hint="We'll only email about launch." />
<Input label="Email" defaultValue="nope" error="Enter a valid email address." />
```

Error copy follows the brand patterns: `Enter a valid email address.`, `This field is required.` — sentence case, period, no exclamation.
